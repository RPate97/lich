# Plan 12 — Skills authoring + CLAUDE.md generator

**Goal:** Author the workflow skills (`change`, `debug`, `onboard`) and reference skills (`prisma`, `hono`, `next`, `tailwind`, `shadcn`, `better-auth`, `vitest`, `playwright`, `turbo`, `lich-cli`) under `tools/cli/templates/v0-stack/.lich/skills/`. Add `lich skills index` to regenerate the project's CLAUDE.md from the skills directory.

**Architecture:**
- Skills are markdown files under `.lich/skills/<category>/<name>.md` with YAML frontmatter (`name`, `description`, `applies-to`).
- `lich skills index` scans the directory, builds an index, and writes CLAUDE.md with section headers per category and a `When to use` summary line per skill.
- Skills are content, not code — the heavy lift is writing them well. Each task is one skill (or a small group).
- Plan 11's template includes the generated CLAUDE.md, so projects start with the full library.
