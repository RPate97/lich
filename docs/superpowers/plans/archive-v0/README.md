# Archived v0 plans (do not use)

**These plans implemented the v0 levelzero system. They are superseded and should not be referenced for v1 work.**

The current plan is at `../2026-05-23-lich-v1-plan-0-foundation.md` (the first of a multi-plan v1 implementation).

## Why this is here

The v0 (`levelzero`) implementation was built across these plans (LEV-5 through LEV-177 epics). The v1 (`lich`) implementation is a new shape — single binary, YAML config, no plugin runtime, no scaffolder. Mixing v0 implementation patterns into v1 work will produce incorrect results.

Files here are kept for git history and so future readers can trace how the codebase evolved. They are NOT guidance for current implementation.

## What's archived

All `2026-05-16-levelzero-*.md` and `2026-05-17-levelzero-*.md` plans (the original 14-plan v0 implementation), `2026-05-16-orchestrator-protocol.md`, `2026-05-16-levelzero-roadmap.md`, and the two dashboard plans from May 21-22.
